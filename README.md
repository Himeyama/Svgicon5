# SVG2ICO
SVG2ICO is the software that convert to ICO from SVG Image.

![Alt text](image.png)

## Install the package
0. Extract `SVG2ICO`
0. Run `onlineInstall.ps1` with PowerShell
	if cannot install the software, try running the below command.

    ```ps1
    Set-ExecutionPolicy RemoteSigned
    ```
    
## Create a package
```ps1
./createPackage
```
